Weather App

This is a 5-day forecast application utilizing HTML, CSS, JavaScript, and the FetchAPI to allow users to search and display the weather for the city of their choosing. The background of the application also changes to a photo that is particular to that city.

Default City | Miami

Upon starting the application, the user will see the default city of Miami with a 5-Day forecast, as well as, a summary of the current weather of the day.

Changing Cities | New York

Users can change cities by entering the city's name, zip code, or nautical coordinates.

Code | Fetch API

Using Open Weather Map's API and Fetch API, I was able to create a weather object and display the weather data in the app.
